# LoveScript
C++ 습작용으로 만든 난해한 언어 LoveScript 사랑해ㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐㅐ

구조는 HQ9+와 brainf를 기반으로 설계됐습니다.

# 명령어
```
I 시작 문
love : Hello, World를 출력한다.
hate : Bye, World를 출력한다.
like : 스택을 1 만큼 증가시킨다.
happy 현 스택을 ASCII 문자로 변환한다.
you 종결문
```
